= Module 1: Set Up Project Directory on Node

In this module you will create a project directory and a README file on the node VM.

== Task 1: Create the project directory

[source,sh,role=execute]
----
ssh node "mkdir -p /home/lab-user/myproject"
----

== Task 2: Create a README file

[source,sh,role=execute]
----
ssh node "echo '# My Project' > /home/lab-user/myproject/README.md"
----

=== Verify

[source,sh,role=execute]
----
ssh node "ls -la /home/lab-user/myproject/"
----

Click *Next* to validate your work. Both tasks must be complete.
