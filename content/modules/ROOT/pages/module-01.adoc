= Module 1: Create a File on the Node VM
:imagesdir: ../assets/images

In this module you will create a file on the node VM from the bastion.

== Exercise: Create the file

. From the *Terminal* tab, SSH to the bastion (it may already be connected).

. Create the required file on the node:
+
[source,sh,role=execute]
----
ssh -o StrictHostKeyChecking=no lab-user@node \
  "mkdir -p /tmp/zt-test && touch /tmp/zt-test/done"
----

=== Verify

[source,sh,role=execute]
----
ssh -o StrictHostKeyChecking=no lab-user@node "ls -la /tmp/zt-test/done"
----

.Expected output
[source,texinfo]
----
-rw-r--r--. 1 lab-user lab-user 0 Mar 30 10:00 /tmp/zt-test/done
----
