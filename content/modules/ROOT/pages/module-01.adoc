= Module 1: Set Up Lab Environment

In this module you will perform tasks on both the node VM and the bastion host.

Use the *Terminal (Node)* tab for node tasks and *Terminal (Bastion)* for bastion tasks.

== Task 1: Create the project directory on node

[source,sh,role="send-to-terminal"]
----
mkdir -p /home/lab-user/myproject
----

== Task 2: Create a README file on node

[source,sh,role="send-to-terminal"]
----
echo '# My Project' > /home/lab-user/myproject/README.md
----

== Task 3: Create a notes file on the bastion

[source,sh,role="send-to-wetty"]
----
touch /home/lab-user/lab-notes.txt
----

=== Verify

[source,sh,role="send-to-terminal"]
----
ls /home/lab-user/myproject/
----

[source,sh,role="send-to-wetty"]
----
ls /home/lab-user/lab-notes.txt
----

Click *Next* to validate your work. All three tasks must be complete.
