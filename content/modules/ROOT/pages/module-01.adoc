= Module 1: Install a Package on the Bastion

In this module you will install the `htop` package on the bastion host.

== Exercise: Install htop

. From the *Terminal* tab, run:
+
[source,sh,role=execute]
----
sudo dnf install -y htop
----

=== Verify

[source,sh,role=execute]
----
rpm -q htop
----

.Expected output
[source,texinfo]
----
htop-3.x.x-x.el9.x86_64
----

Click *Next* to validate your work.
