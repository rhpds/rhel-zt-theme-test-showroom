= Module 2: Install a Package on the Node

In this module you will install the `htop` package on the node VM.

== Exercise

From the *Terminal* tab, install htop on the node:

[source,sh,role=execute]
----
ssh node "sudo dnf install -y htop"
----

=== Verify

[source,sh,role=execute]
----
ssh node "rpm -q htop"
----

.Expected output
[source,texinfo]
----
htop-3.x.x-x.el9.x86_64
----
