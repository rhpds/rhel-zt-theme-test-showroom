= Module 2: Install a Package on the Node

In this module you will install the `nc` package on the node VM.

== Exercise

From the *Terminal* tab, install nc on the node:

[source,sh,role=execute]
----
ssh node "sudo dnf install -y nc"
----

=== Verify

[source,sh,role=execute]
----
ssh node "rpm -q nc"
----

.Expected output
[source,texinfo]
----
nc-3.x.x-x.el9.x86_64
----
