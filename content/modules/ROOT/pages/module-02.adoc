= Module 2: Create a Directory on the Node VM

In this module you will create a project directory on the node VM from the bastion.

== Exercise: Create the directory

. From the *Terminal* tab, SSH to the node and create the directory:
+
[source,sh,role=execute]
----
ssh node "mkdir -p /home/lab-user/myproject"
----

=== Verify

[source,sh,role=execute]
----
ssh node "ls -la /home/lab-user/ | grep myproject"
----

.Expected output
[source,texinfo]
----
drwxr-xr-x. 2 lab-user lab-user 6 ... myproject
----
