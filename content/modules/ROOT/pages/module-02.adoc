= Module 2: Ensure sshd Is Running on the Node
:imagesdir: ../assets/images

In this module you will ensure the `sshd` service is active on the node VM.

== Exercise: Check and start sshd

. Check the current status:
+
[source,sh,role=execute]
----
ssh -o StrictHostKeyChecking=no lab-user@node "systemctl status sshd --no-pager"
----

. If not running, enable and start it:
+
[source,sh,role=execute]
----
ssh -o StrictHostKeyChecking=no lab-user@node "sudo systemctl enable --now sshd"
----

=== Verify

[source,sh,role=execute]
----
ssh -o StrictHostKeyChecking=no lab-user@node "systemctl is-active sshd"
----

.Expected output
[source,texinfo]
----
active
----
