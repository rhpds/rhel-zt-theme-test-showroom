== How the runner works (RHEL mode)

The `zt_runner_rhel` role adds a runner container to the showroom pod.
The runner SSHes to the bastion using `BASTION_HOST`/`ANSIBLE_PASSWORD` env vars,
then the Ansible playbooks in `runtime-automation/` connect to the node from there.
