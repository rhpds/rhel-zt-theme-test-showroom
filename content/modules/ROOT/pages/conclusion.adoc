== How the ZT grader worked

When you clicked *Validate*, the ZT runner:

. SSH'd to the bastion using `BASTION_HOST` and `ANSIBLE_PASSWORD` env vars
. From the bastion, checked the node via SSH
. Reported pass or fail back to this UI

When you clicked *Solve*, the runner performed the task for you automatically via the same SSH path.
